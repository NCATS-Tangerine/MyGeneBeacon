This is http://dev.mygene.info, the dev site for v2 MyGene.info API.

A few example queries are like these:

  • http://dev.mygene.info/gene/1017
  • http://dev.mygene.info/gene/1017?fields=name,symbol,refseq
  • http://dev.mygene.info/gene/1017?callback=jsonpcallback
  • http://dev.mygene.info/query?q=CDK2
  • http://dev.mygene.info/query?q=cdk?
  • http://dev.mygene.info/query?q=refseq:NM_001798
